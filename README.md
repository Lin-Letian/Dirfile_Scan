# Dirfile_Scan
Dirfile Scan-目录/文件爆破工具

图形化运行配置界面，可设置代理

可自定义对 #目录 形式爆破

自定义已知Web跟目录或路径，可对该路径之上进行二层爆破

对网站目录进行自动爬取，对爬取后的路径进行字典爆破，不限层数

可自定义设置爆破间隔、线程、超时、请求方式(GET\HEAD\POST)\GET、响应状态

设置代理时，可以通过Check Proxy按钮获取外网地址

内置143个作者亲选字典，扫描率高，也可自定义字典。
